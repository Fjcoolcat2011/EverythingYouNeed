Update Mode

Turn on Update Mode to close the shop to everyone. A full-screen page appears on every device instead of the shop. Staff can still access the shop by typing the unlock code at the bottom of the update page, or by opening the Owner Panel.

Backup & Restore

Orders, items, accounts, tickets, and announcements are synced through your cloud database. Export saves a snapshot of everything to a JSON file you can keep as a backup. Import overwrites the cloud data with that file.
